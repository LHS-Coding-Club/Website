What is Coding?

When you do coding, you write "code," which is just a set of directions that tell a computer what to do. Everything digital is powered by code, whether it be a digital alarm clock, or a supercomputer. Learning how to code will let you make computers do want you want them to.

Do I have to know coding to join?

Nope, whether you're a coding wizard, professional scratch programmer, or have never done any sort of coding before, you can still join. In fact, most of our members had never done any coding before they joined.
